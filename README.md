# MPIS平台

## 使用

- 关闭git自动转换换行符

  ```bash
  git config --global core.autocrlf false
  ```

- 克隆代码

  ```bash
  git clone https://gitee.com/sinoso/mpis.git
  ```

- vs code安装``editorconfig``、``prettier``、``Path Intellisense``、``Npm Intellisense``插件

- 安装开发依赖

  ```bash
  yarn install
  ```

- 开发

  ```bash
  yarn start
  ```

- 编译

  ```bash
  yarn build
  ```
